# Protótipo - Gestão Fazenda (Web)

Protótipo dinâmico de aplicação web desenvolvido como entrega de atividade do módulo.
Tecnologias usadas (CDN prototyping):
- React 18 (via UMD CDN)
- Tailwind CSS (CDN)
- LocalStorage para persistência simples

Arquivos:
- index.html — protótipo funcional (colocar no navegador para testar)

Como executar:
1. Baixe os arquivos e abra o `index.html` no navegador.
2. O protótipo funciona offline (após carregar CDNs, pode exigir conexão inicial).
3. Para publicar no GitHub Pages, crie um repositório e envie `index.html` (e assets). Ative GitHub Pages em branch `main` com root.

Sugestão de commit / mensagem do repositório:
`feat: protótipo inicial - React + Tailwind (single-file)`
